Introduction

Excitons–Coulomb-bound electron-hole pairs in semiconductors and insulators–are canonical 2B quantum systems. Their formation, binding energies, and radiative recombination govern key optical and electronic responses and underpin technologies ranging from light-emitting diodes to photovoltaics . In 2D materials, reduced dielectric screening and quantum confinement amplify excitonic effects, yielding large binding energies and rich many-body phenomena confirmed by precision spectroscopy in monolayer TMDs . Accurate modeling, therefore, requires methods that move beyond elementary approximations and incorporate material-specific screening and band-structure inputs. The LS equation provides an integral-equation formulation equivalent to the Schrödinger eigenproblem with appropriate boundary conditions. For 2D excitons, both the bare Coulomb interaction and the Rytova-Keldysh (RK) potential are widely used, the latter capturing nonlocal, environment-dependent screening characteristic of atomically thin layers and their substrates . Solving the LS integral equation in momentum space yields binding energies and wave functions and enables controlled studies of dielectric constants, screening lengths, and effective masses under experimentally relevant conditions. Momentum space LS techniques have been successfully applied to diverse 2B bound states and provide a flexible route for benchmarking and extension . To provide free, broad access, ensure reproducibility, and support teaching at scale, we introduce a free, interactive online platform that unifies theory, numerics, and visualization in a single browser-based venue-no installation required and usable across devices-thereby removing platform barriers and making LS workflows openly available to students and researchers alike. In this work, we introduce an open-access, open-source website for solving the LS integral equation for excitonic systems in 2D momentum space, available at https://csu-physics.github.io/2B_2D/. The platform couples concise theoretical primers with browser-executable Jupyter notebooks (JupyterLite/Pyodide) to provide end-to-end workflows from parameter setup and quadrature discretization to solution, self-consistency checks via Hamiltonian expectation values, convergence studies, and publication-ready visualization. Core capabilities include (i) computing bound state spectra and wave functions for Coulomb and RK interactions; (ii) interactive parameter scans with immediate feedback; and (iii) a modular, extensible design, so kernels and models for 2D quantum systems can be adapted or extended with minimal effort. Designed for advanced undergraduate and graduate instruction as well as research prototyping, the site lowers the barrier to LS-based exciton calculations by unifying transparent methods, reusable components, and intuitive interactivity in a single, freely accessible resource. To validate the formalism and support instruction, the website also includes pedagogical testbeds that solve the LS equation for 2B problems: a hydrogenic system (Coulomb interaction) and a schematic deuteron bound state (e.g., Yukawa-type/Malfliet-Tjon (MT) interactions). These modules serve as clear numerical benchmarks and classroom assets before the same toolkit is applied to excitons using the RK interaction.

Lippmann–Schwinger Equation for Two-Body Bound States in 2D

The starting point for a 2B bound state with reduced mass \(\mu\) is the time-independent Schrödinger equation \[(H_0 + V)\,|\Psi\rangle = E\,|\Psi\rangle, \label{eq:sch}\] where \(H_0\) is the free Hamiltonian, \(V\) is the interaction, \(E<0\) is the binding energy, and \(|\Psi\rangle\) is the corresponding bound state wave function. Rearranging Eq. [eq:sch] gives the homogeneous LS equation for a bound state \[|\Psi\rangle = G_0(E)\,V\,|\Psi\rangle, \qquad G_0(E) \equiv \frac{1}{E - H_0}, \label{eq:ls_operator}\] which expresses the state in terms of the free propagator \(G_0(E)\) and the interaction. In a momentum basis \(|{\mathbf p}\rangle\), adopting the completeness \(\int \! d^2 p\, |{\mathbf p}\rangle\langle {\mathbf p}| = \mathds{1}\), the LS equation projects to \[\psi({\mathbf p}) = \frac{1}{E - \dfrac{p^2}{2\mu}}\, \int_0^\infty \! dp'\, p' \int_0^{2\pi} \! d\phi'\, V({\mathbf p},{\mathbf p}')\, \psi({\mathbf p}'), \label{eq:LS_vector}\] where \(\psi({\mathbf p}) \equiv \langle {\mathbf p}| \Psi\rangle\) and \(V({\mathbf p},{\mathbf p}') \equiv \langle {\mathbf p}| V | {\mathbf p}' \rangle\). For central interactions in 2D, it is convenient to separate the angular dependence by a partial-wave (PW) expansion, which reduces Eq. [eq:LS_vector] to a one-dimensional integral equation for each angular-momentum channel \(m\) \[\psi_m(p) = \frac{1}{E - \dfrac{p^2}{2\mu}} \int_0^\infty \! dp'\, p'\, V_m(p,p')\, \psi_m(p'). \label{eq:LS_pw}\] The PW-projected kernel can be obtained from the angular integration in momentum space, \[\begin{aligned} V_m(p, p') = \int_0^{2\pi} d\phi' \, V(p, p', \phi') \cos(m \phi') \equiv \int_0^\infty dr r \, J_m(pr) V(r) J_m(p'r). \end{aligned}\] with \(J_m\) the Bessel function of the first kind. The PW representation for each \(m\) reduces the 2D integral equation to a one-dimensional radial form. This framework is used throughout to compute 2D bound states, e.g., hydrogenic systems with Coulomb interactions and the deuteron with Yukawa-type interactions, and, in particular, excitons with the RK interaction, within a momentum-space LS scheme.

A convenient internal consistency check for a numerically obtained 2B binding energy and wave function is to compare the eigenvalue \(E\) from the LS equation with the Hamiltonian expectation value, \(\langle H \rangle = \langle H_0 \rangle + \langle V \rangle\), evaluated using the corresponding momentum-space wave function. The degree of agreement between \(E\) and \(\langle H \rangle\) provides a direct measure of the numerical accuracy of the method. For a given PW channel \(m\), with the wave function normalization \[\int \! d^2p\, \psi^2({\mathbf p}) = 2\pi \int_0^\infty \! dp\, p\, \psi_m^2(p) = 1. \label{eq:PW_norm}\] The Hamiltonian expectation value, expressed in terms of the kinetic and potential contributions, reads \[\begin{aligned} \langle H \rangle &= \langle H_0 \rangle + \langle V \rangle, \label{eq:H_split} \\[3pt] \langle H_0 \rangle &= 2\pi \int_0^\infty \! dp\, p\; \frac{p^2}{2\mu}\, |\psi_m(p)|^2, \label{eq:H0_exp} \\[3pt] \langle V \rangle &= 2\pi \int_0^\infty \! dp\, p \int_0^\infty \! dp'\, p'\; \psi_m(p)\, V_m(p,p')\, \psi_m(p') \,. \label{eq:V_exp} \end{aligned}\]

Numerical Implementation

To numerically solve the LS integral equation [eq:LS_pw] for a 2B bound state in 2D, represented in the PW framework, the integral equation should be transformed into a discretized form that can be solved iteratively or through matrix diagonalization. To efficiently handle the integrals, variable transformations are introduced to map the infinite domains of momentum \(p\), spatial coordinate \(r\), and angle \(\phi\) onto finite intervals. The transformations are defined as: \[p,p', r = \frac{1+x}{1-x}, \quad \phi' = \pi (1+x), \quad x \in [-1, 1],\] where \(x\) is a variable spanning the interval \([-1, 1]\), suitable for integration using Gaussian-Legendre quadrature. The discretized form of the integral equation for the wave function \(\psi_m\) becomes an eigenvalue problem: \[\lambda \ \psi_m = {\cal K}(E) \, \psi_m,\] where \({\cal K}(E)\) represents the integral kernel operator dependent on the binding energy \(E\). The eigenvalue \(\lambda\) is used as a criterion to verify the solution. The binding energy \(E\) is determined by solving for the eigenvalue \(\lambda = 1\). This involves iteratively searching over initial guesses for \(E\) until convergence is achieved. A tolerance of \(10^{-6}\) is set for the eigenvalue condition, ensuring high accuracy in the computed energy.
The iterative search procedure can be outlined as follows:

  1. Select two initial guesses for the binding energy E.

  2. Construct the kernel \({\cal K}(E)\) for each chosen energy.

  3. Solve the eigenvalue problem \(\lambda \, \psi_m = {\cal K}(E) \, \psi_m\) to obtain \(\lambda\) and the corresponding wave function \(\psi_m\) for both initial guesses.

  4. Using these two initial pairs \((E, \lambda)\), perform a linear fit to predict the next energy value for which \(\lambda = 1\).

  5. Iterate the process with the updated energy values until \(\lambda\) converges to \(1\) within the specified tolerance.

The kernel \({\cal K}(E)\) is diagonalized at each step, which allows for the extraction of both the eigenvalue \(\lambda\) and the wave function \(\psi_m\). Efficient numerical techniques such as matrix diagonalization, implemented through libraries like LAPACK, or iterative solvers like the Lanczos algorithm , can be employed for this purpose.

Systems Studied

To ensure the robustness and accuracy of our computational framework, a critical first step is validating the numerical methods against established benchmarks. For this purpose, we compare our simulated results for the 2D hydrogenic atom and the 2D deuteron system with well-known theoretical findings. These systems serve as ideal case studies: the hydrogenic atom, with its fundamental Coulomb potential, provides an exact analytical benchmark for bound states, while the deuteron, modeled with the more intricate MT potential, demonstrates the framework’s ability to handle complex nuclear interactions. By showing strong agreement between our computational outputs and trusted reference values, we establish confidence in the accuracy and reliability of our algorithms. This rigorous validation provides the essential foundation for applying our methods to more complex systems, such as excitons in 2D materials, where the RK potential is used to probe the influence of screening length and dielectric constants on binding energies and wavefunctions.

Hydrogenic atom in 2D

We begin with the 2D hydrogenic system, a well-established case that provides an ideal benchmark for testing the accuracy of our numerical method. In contrast to its three-dimensional analogue, the 2D system exhibits distinctive features in its energy spectrum and wave function behavior arising from the reduced dimensionality. In this section, we present both analytical and numerical treatments of the problem, working in natural units where \(\hbar c = e = m = 1\). The 2B potential energy between a nucleus of charge \(Z\) and a single electron in configuration space is given by \[V(r) = -\dfrac{Z}{r},\] which can be expressed in momentum space as \[V({\mathbf p}, {\mathbf p}') = \dfrac{-Z}{2\pi \vert {\mathbf q}\vert}, \quad {\mathbf q}= {\mathbf p}- {\mathbf p}'.\] The analytical solutions of the LS integral equation for the 2B binding energy levels are given in Ref. \[\label{Eq.E_Hydrogen_exact} E_{exact} = -\dfrac{Z^2 m}{4\left(n + 1/2\right)^2}, \quad n = 0, 1, 2, \ldots\] The exact energy levels for 2D hydrogenic atoms offer a clear reference for comparison. Evaluating the agreement between our computed values and the known analytical results allows us to assess the reliability of the approach and identify potential issues at an early stage. Establishing this validation is essential before extending the method to more complex systems, such as excitons or deuterons. The results of this comparison are presented in Table 1, showing that our numerical values closely match the exact energy levels of the 2D hydrogenic atom. The relative errors are very small, less than 0.4% for the ground state and the first three excited states, demonstrating the accuracy and reliability of our method.

Excitons in 2D Materials

Having validated the framework on 2D hydrogenic atoms and the deuteron, we now apply it to excitons-bound electron-hole pairs formed by Coulomb attraction that play a central role in the optical response of 2D materials. In this section we solve the LS integral equation for excitons using the RK potential. This yields exciton binding energies and wave functions, providing insight needed to predict exciton behavior in practical devices and to inform the design of materials with targeted optical properties. The RK potential accounts for nonlocal, distance-dependent dielectric screening in 2D materials, and thus models electron-hole interactions more accurately than a bare Coulomb potential. The explicit forms of the RK potential in configuration and momentum space are \[\begin{aligned} V(r) &=& -\dfrac{\alpha \hbar c \pi}{2 r_0 \kappa}\left[H_0\left(\dfrac{r}{r_0}\right)-Y_0\left(\dfrac{r}{r_0}\right)\right], \cr % V({\mathbf p},{\mathbf p}') &=& - \dfrac{1}{4\pi^2} \left( \dfrac{1}{4\pi \epsilon_0 \kappa} \dfrac{2\pi e^2}{q(1 + r_0 q /\kappa )}\right), \quad q = | {\mathbf p}- {\mathbf p}' |, \end{aligned}\] where \(H_0\) and \(Y_0 denote\), respectively, the order-zero Struve function and the Bessel function of the second kind. Here \(r\) denotes the electron-hole separation, \(r_0\) is the screening length, and \(\kappa=(\epsilon_1+\epsilon_2)/2\) is the average environmental dielectric constant. The factor \(\epsilon_0/e^2=\big(4\pi\,\alpha\,\hbar c\big)^{-1}\) converts between the fine-structure constant and energy units. We use \(\alpha^{-1}=137.035999084\), \(\hbar c=1973.269804\ \text{eV}\!\cdot\!\text{\AA}\), and the electron rest mass \(m_0=0.510998950\ \text{MeV}\). User Guide: Accessing, Editing, and Running the Interactive Code

The website hosts runnable Jupyter notebooks (via JupyterLite/Pyodide), so you can explore 2B quantum bound state problems in 2D directly in your browser-no installation required.

Getting In

  1. Navigate to https://csu-physics.github.io/2B_2D/.

  2. From the homepage, select a notebook from the top menu bar (e.g., Hydrogen in 2D, Deuteron in 2D, Exciton in 2D).

  3. Allow a few seconds for the Python runtime to load.

  4. From the file browser in the left sidebar, open a Jupyter notebook (e.g., Hydrogen\(_-\)2D.ipynb, Deuteron\(_-\)2D.ipynb, Exciton\(_-\)2D.ipynb).

Visual cue: You will see the Jupyter toolbar (Run, Kernel, etc.) and executable code cells.

Saving or Resetting Your Work

  • Download your edits: Right-click a notebook (i.e. Exciton\(_-\)2D.ipynb) and choose Download.

  • Reset to original: Refresh the page (sessions are ephemeral).

Troubleshooting

  • Blank or unresponsive: Ensure JavaScript is enabled; refresh the page.

  • Memory or speed issues: Reduce Np or Nphi; close heavy tabs.

  • Stuck kernel: Kernel \(\rightarrow\) Restart Kernel.

  • No changes after edits: Run \(\rightarrow\) Run All Cells.

Quick Checklist

  1. Open a notebook and Run All once.

  2. Edit Np, Nphi, m, and potential parameters.

  3. Restart Kernel and Run All Cells.

  4. Confirm convergence of energy; check \(\langle H\rangle\) vs. \(E\) (small relative percentage difference); inspect wave-function plots.

  5. Download the notebook or images if you wish to keep results.

Conclusion

We have released an open, browser-based platform for solving the Lippmann-Schwinger equation in two dimensions at https://csu-physics.github.io/2B\(_-\)2D/. The website delivers executable Jupyter notebooks (via JupyterLite/Pyodide) that implement the full workflow, from parameter setup and quadrature discretization to numerical solution and binding energy calculations, validation via the Hamiltonian expectation value, and visualization of wave functions. Case studies for the 2D hydrogenic atom (Coulomb), the deuteron (Malfliet-Tjon), and excitons (Rytova-Keldysh) provide ready-to-run, momentum-space implementations that connect theory with computation. Hosting on GitHub Pages ensures free, immediate access without local installation, while the open repository invites community review and contributions (issues and pull requests). This model promotes reproducibility, inclusivity, and rapid iteration on methods, examples, and documentation. For education, the platform supports active learning with edit-and-run examples, convergence studies, and interpretive plots that help students bridge abstract quantum concepts with concrete numerical results. For research, it offers a transparent, reproducible environment for prototyping kernels, testing discretizations, and benchmarking bound state solvers in 2D, lowering the barrier to entry for new investigations of excitons and related 2B problems in atomically thin materials.

Website Development and Technical Details

The interactive platform for solving the LS integral equation in 2D momentum space is hosted on GitHub Pages at https://csu-physics.github.io/2B\(_-\)2D/, providing free, browser-based access with no local installation. Hosting on GitHub ensures version control, transparency, and community contributions through issues and pull requests, enabling continuous improvement (e.g., new case studies or numerical schemes) while keeping the resource openly accessible. The website is organized as executable Jupyter notebooks (Python) that implement the full workflow: importing required libraries (e.g., NumPy/SciPy for numerics and Matplotlib for plots), setting initial parameters (physical constants, material properties, grid extents, and quadrature orders), applying Gauss-Legendre quadrature to discretize the continuous variables (relative distance, relative momentum, and angular coordinates), assembling and solving the LS integral equation, calculating binding energies, visualizing wave functions, and performing parameter sweeps. The learning path includes a brief refresher on the Schródinger equation and the LS equation in momentum space, with ready-to-run 2D implementations. Case studies demonstrate the framework for a 2D hydrogenic atom (Coulomb), a 2D deuteron (Malfliet-Tjon), and a 2D exciton (Rytova-Keldysh).

Front-End Implementation

Overview of Website Interface and User Interaction

The website was designed to be easy to use and focused on giving users an interactive experience. The layout of the website is simple and intuitive, allowing users to easily navigate the different sections, and interactive tools are built right into the pages, meaning users can run code, adjust parameters, and immediately see results, all without needing to leave the website. This design philosophy aims to minimize cognitive load and maximize engagement, allowing students to focus on the physics concepts rather than the mechanics of the platform.

Description of Interactive Components, Visualization Tools, and User Controls

The website’s interactivity is delivered through embedded Jupyter notebooks that run entirely in the browser via JupyterLite (Python kernel powered by Pyodide). Users can edit and execute code on the page, adjust model inputs (e.g., potential parameters, and the number of mesh points for momentum/angle discretization), and immediately recompute binding energies and wave-function plots. User controls and visualization: Inputs are displayed as code cells in the Jupyter Notebook, and results update on run. Plots rendered inline using Matplotlib, enabling quick comparisons and exploratory analysis. Environment and workflow: No installation is required; notebooks load in seconds and run fully client-side. Users can open hosted examples, modify them, and see results in real-time. The environment supports multiple kernels and lightweight consoles for rapid experiments, providing a smooth, self-contained workflow for learning and exploration.

Technologies Used

The website’s frontend is built with Next.js (React on Node.js), which provides routing, build tooling, and both server-side rendering (SSR) and static-site generation (SSG) for fast, reliable delivery. After implementing the core UI in Next.js, we integrated the interactive computation environment by embedding hosted Jupyter notebooks in iframes. This approach cleanly separates the web application (navigation, layout, performance) from the notebook runtime (code execution and visualization), while allowing the interactive components to appear seamlessly within the website’s pages.

We host the website on GitHub Pages for a simple, no-cost, and tightly integrated workflow. Changes pushed to the repository trigger an automatic build and deploy (via GitHub Pages/GitHub Actions), so the website stays current without manual steps or separate infrastructure. Because the website is served as static HTTPS content with CDN caching, it loads quickly and reliably, and requires no application server or container orchestration. This lightweight deployment model keeps maintenance minimal and aligns with our goal of providing an easily accessible, always-available educational resource.

Technical Overview of Jupyter Notebook Usage

We build the interactive notebooks with JupyterLite and host them alongside the website, then embed each notebook into the Next.js pages via <iframe> so students can open, edit, and run code directly in the browser-no installs required. JupyterLite provides a full JupyterLab interface that runs entirely client-side: a Python kernel powered by Pyodide (WebAssembly) executes in a Web Worker, enabling numerical computing, plotting, and widget-based interactivity within the page. For this project we started from an open-source JupyterLite template (GitHub) and tailored it to our content. To keep sessions clean and secure, we disabled persistence, so any edits exist only for the current session and are not saved to the server. Notes on capabilities and limits: Browser-based Python is ideal for teaching and demos but has constraints: available memory is limited, package availability is narrower than a full desktop Python, filesystem access is sandboxed, and heavy workloads may run more slowly than native execution. Despite these trade-offs, JupyterLite delivers a fast, zero-setup environment that’s well-suited to our instructional goals.
